PROJETO CRUD - MOBILE

NECESSÁRIO: 
-FIREBASE
-EXPO
-VISUAL CODE (SE NÃO OPTAR PELO CMD)

1- BAIXE OS ARQUIVOS E DESCOMPACTE-OS EM UMA PASTA DE NOME QUALQUER;

2- CRIE UMA CONTA E UM BANCO DE DADOS NO 'FIREBASE';

3- MUDE OS DADOS DE 'var firebaseConfig' PARA OS DADOS DA SUA CONTA FIREBASE;

4- USANDO O 'CMD EM MODO DE ADMINISTRADOR', INSTALE O 'EXPO' COM O COMANDO 'npm install --global expo-cli';

5- AINDA USANDO O CMD, ACESSE A PASTA ONDE ESTÃO OS ARQUIVOS DO PROJETO E EXECUTE O COMANDO 'npm install'. SE HOUVER ALGUM ERRO, EXECUTE O COMANDO 'npm install --force';

6- INSIRA DADOS NO FIREBASE DE MODO QUE OS CAMPOS DOS 'DOCUMENTOS' (COLUNAS) SEJAM 'name', 'email', 'phone', 'curso';

7- AINDA NA PASTA RAÍZ DO PROJETO, EXECUTE O COMANDO 'expo start'. ASSIM QUE O PROJETO CARREGAR, PRESSIONE 'w' PARA ABRIR O PROJETO NO NAVEGADOR; 

8- AGORA O PROJETO DEVERÁ FUNCIONAR CORRETAMENTE.
